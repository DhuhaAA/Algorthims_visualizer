# Algorithm Race: Sorting Visualizer

Interactive side-by-side comparison of sorting algorithms with real-time visualization.

## Features
- Compare Bubble, Merge, Quick, and Radix Sort
- Adjustable array sizes and input patterns
- Live performance metrics
- Educational algorithm encyclopedia with Python implementations

## Run Locally
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
